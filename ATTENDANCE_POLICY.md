## Typical Workflow

### Morning Routine (Admin/HR):

1. **Open Admin Application**
2. **Navigate to Daily Attendance**
3. **As employees arrive:**
   - Use face recognition kiosk, OR
   - Manually mark check-in for each employee

### Throughout the Day:

- Monitor attendance dashboard
- Mark lunch breaks (if tracked)
- Handle late arrivals
- Process attendance corrections

### End of Day:

- Mark check-outs for employees
- Review daily attendance report
- Handle any discrepancies
- Generate daily report

## Face Recognition Kiosk Setup

### Hardware Required:
- Tablet or computer with camera
- Mounted at office entrance
- Internet connectivity
- Admin credentials

### Setup Process:

1. **Register Employee Faces** (Admin App)
   - Navigate to Face Registration
   - Select employee
   - Capture multiple face angles
   - Save face data

2. **Setup Kiosk Device**
   - Login to Admin App
   - Navigate to Face Attendance
   - Position device at entrance
   - Keep app open and running

3. **Employee Usage**
   - Employee stands in front of camera
   - System recognizes face
   - Attendance marked automatically
   - Confirmation displayed

## Attendance Corrections

### When Corrections are Needed:

- Employee forgot to check in/out
- Face recognition failed
- System was down
- Late arrival with valid reason
- Early departure approved

### How to Make Corrections:

1. **Employee notifies HR/Admin**
   - Via email or request form
   - Provides reason and correct time
   
2. **Admin/HR verifies request**
   - Checks validity
   - Verifies with manager if needed
   
3. **Admin makes correction**
   - Opens attendance record
   - Updates times
   - Adds remarks/notes
   - Saves changes

4. **Employee can view updated record**
   - Sees corrected time in app
   - History shows update

## Leave Management Integration

### How Leaves Affect Attendance:

- **Approved Leaves** automatically marked as LEAVE status
- **Pending Leaves** don't affect attendance until approved
- **Rejected Leaves** require attendance to be marked
- **Half-day Leaves** marked as HALF_DAY status

### Process:

1. Employee applies for leave (Employee App)
2. Admin/HR approves/rejects (Admin App)
3. If approved, attendance automatically marked as LEAVE
4. No manual marking needed for leave days

## Troubleshooting

### Common Issues:

**Issue**: Employee attendance not showing in app
- **Solution**: Check if admin has marked it; refresh app; wait a few minutes for sync

**Issue**: Face recognition not working
- **Solution**: Use manual marking; update face data; check camera; check lighting

**Issue**: Incorrect time recorded
- **Solution**: Submit correction request to admin/HR with correct time and reason

**Issue**: Attendance marked as absent but employee was present
- **Solution**: Contact admin immediately; provide proof (email, meeting records, etc.)
